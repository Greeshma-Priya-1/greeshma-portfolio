# Portfolio Preview

This project is a static HTML portfolio. You do **not** need Python or a virtual environment to view it.

## How to view
- Quick open: double-click `index.html` or use VS Code `View in Browser`/`Open with Live Server`.
- Recommended: install the **Live Server** extension (Ritwick Dey), right-click `index.html`, choose "Open with Live Server" for auto-reload.

## Notes
- Do not launch `index.html` with a Python debugger; use the browser launch config in `.vscode/launch.json` instead.
- If PowerShell blocks `.venv\Scripts\Activate.ps1`, you can ignore it for this static site. Activation is not required unless you add backend code later.
